Vesture

Vesture is a simple outfit tracking app built to help users understand how they actually use their wardrobe. It lets you add clothing items with photos, log daily outfits in a few taps and view a clean history of everything you’ve worn. The app also generates basic analytics, showing your most and least worn pieces so you can make more intentional wardrobe choices.

Features

Wardrobe Catalog

Add clothing items with photos

Organize items by category, including tops and bottoms

Subcategories for shirts, t-shirts, jerseys, pants, shorts and more


Daily Outfit Logging

Select a top and bottom to quickly log the day’s outfit

Edit or update today’s entry when needed

Automatically adds to your wardrobe history


History View

Scroll through a visual timeline of past outfits

Each entry shows the exact items worn on that date

Helps track style changes and usage patterns


Wear Statistics

Most worn items

Least worn items

Wear counts for each piece

Simple charts highlighting your top used clothing


UI and Experience

Clean, minimal layout designed for daily use

Fully supports light and dark theme

Fast workflow focused on reducing friction


Tech Stack

Next.js (App Router)

TypeScript

Tailwind CSS

Firebase (Auth, Firestore, Storage)

Recharts for statistics visualizations


Why Vesture?

Vesture aims to make wardrobe tracking effortless. Instead of overthinking outfits or buying things you don’t need, the app gives you a small amount of structure and insight, all without feeling like a chore. Add your clothes once, log your day in a second and let the data handle the rest.

Getting Started

1. Clone the repository:

git clone https://github.com/amharies/Vesture


2. Install dependencies:

npm install


3. Add your .env file with Firebase config.


4. Run the dev server:

npm run dev
